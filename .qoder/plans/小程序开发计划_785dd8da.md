# 梵优茗宠小程序开发计划

## 技术选型

- **框架**: uni-app (Vue3 + Composition API)
- **构建工具**: Vite
- **状态管理**: Pinia
- **UI 组件库**: uni-ui + 自定义组件
- **CSS 预处理**: SCSS
- **HTTP 请求**: 封装 uni.request，统一 Token 拦截
- **目标平台**: 微信小程序 / H5 / 支付宝 / 抖音小程序

## 服务端 API 端点汇总

| 模块 | 路径前缀 | 核心接口 |
|------|---------|---------|
| 用户认证 | `/app/user` | wxLogin, getProfile, updateProfile |
| 地址管理 | `/app/user/address` | CRUD |
| 宠物管理 | `/app/user/pet` | CRUD |
| 商品 | `/app/product` | list, detail, flash |
| 商品分类 | `/app/product-category` | tree |
| 组合包 | `/app/bundle` | list, detail |
| Banner | `/app/banner` | active |
| 同城商家 | `/app/merchant` | list, detail, apply |
| 购物车与订单 | `/app/order` | cart CRUD, order CRUD |
| 配餐 | `/app/meal` | quiz, plans, order |
| 课程 | `/app/course` | list, detail, purchase, my |
| 分佣 | `/app/commission` | account, logs, invites, withdraw |
| 财务 | `/app/finance` | points, recharges, consumptions, sign-in |
| 售后 | `/app/refund` | submit, list, detail |
| 直播间 | `/app/live-room` | list |

## Task 1: 项目初始化与基础架构

**提交信息**: `feat(applet): init uni-app project with base architecture`

- 使用 `npx degit dcloudio/uni-preset-vue#vite-ts` 初始化 uni-app Vue3 项目到 `applet/` 目录
- 配置 `manifest.json` (appid、多端配置)
- 配置 `pages.json` (页面路由、tabBar 5个tab：首页/同城/轻创/购物车/我的)
- 配置环境变量 (开发/生产 API 地址)
- 搭建目录结构:
  ```
  applet/
  ├── src/
  │   ├── api/          # 接口层
  │   ├── components/   # 公共组件
  │   ├── composables/  # 组合式函数
  │   ├── pages/        # 页面
  │   ├── store/        # Pinia 状态管理
  │   ├── static/       # 静态资源
  │   ├── styles/       # 全局样式
  │   ├── utils/        # 工具函数
  │   ├── App.vue
  │   ├── main.ts
  │   └── manifest.json
  ├── package.json
  └── vite.config.ts
  ```
- 封装 HTTP 请求工具 (`utils/request.ts`)：Token 自动注入、401 跳转登录、统一错误处理
- 初始化 Pinia store 结构
- 创建全局 SCSS 变量文件 (参照 design/style.css 的色值体系)
- 安装依赖并验证编译通过
